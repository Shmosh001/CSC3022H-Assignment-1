Files: 
	Driver.cpp - Interface of program with menu options that call the StudentDatabase class. This program will run on start.
	StudentDatabase.cpp - Stores info from a database using methods that are called from the Driver class. It also allows search for a student, adding a student and displaying their average grade. 
	StudentDatabase.h - holds declarations of StudentDatabase for other programs to use.
	makefile

Instructions:
1. Type 'make' and press return
2. Type 'make run' and press return to run program
3. When finished with the program type 'make clean' and press return

Database Format:
FirstName#Surname#StudentNumber#Grades
Student Number is all uppercase and grades are seperated by spaces
